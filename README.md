## Installation

### Voraussetzungen

- Filestash Source (Commit `272eb29f`)
  ```bash
  git clone https://github.com/mickael-kerjean/filestash.git
  cd filestash && git checkout 272eb29f
  ```
- Go 1.26+
- Make

### 1. Patch-Dateien kopieren

```bash
cd filetash-patch
cp src/modal_move.js /path/to/filestash/public/assets/pages/filespage/
cp src/ctrl_submenu.js /path/to/filestash/public/assets/pages/filespage/
cp src/model_files.js /path/to/filestash/public/assets/pages/filespage/
cp src/modal.css /path/to/filestash/public/assets/components/
cp src/index.frontoffice.html /path/to/filestash/public/
```

### 2. Syntax-Check (empfohlen)

```bash
node --check public/assets/pages/filespage/modal_move.js
node --check public/assets/pages/filespage/ctrl_submenu.js
node --check public/assets/pages/filespage/model_files.js
```

### 3. Build

```bash
cd filestash && make build 2>&1
```

### 4. Deploy

```bash
scp dist/filestash root@<HOST>:/tmp/filestash-new
ssh root@<HOST> "docker cp /tmp/filestash-new filestash:/app/filestash && docker restart filestash"
```

## Verwendung

### VERSCHIEBEN-Button

1. Datei/Ordner auswählen
2. "Verschieben"-Button klicken
3. Zielordner im Modal auswählen (Breadcrumb + ▶-Navigation)
4. OK bestätigen

**Statusmeldung:** "`'filename' wurde verschoben`"

### Keyboard Shortcuts

| Taste | Funktion |
|-------|----------|
| **F2** | Umbenennen (1 Datei) |
| **Del** | Löschen (1+) |
| **M** | Verschieben (1+) |
